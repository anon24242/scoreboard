# **App Name**: Cricket Central

## Core Features:

- Score Entry (Admin): Admin page to input match data (teams, scores, overs, wickets).
- Real-time Updates: Display real-time scores and match updates to users.
- User Match View: Display selected details of the current match to end-users, emphasizing readability.
- Data Storage: Store match data in a database for later retrieval and statistics.

## Style Guidelines:

- Primary color: Forest green (#228B22) evoking the cricket field.
- Background color: Off-white (#F5F5DC) providing a clean backdrop.
- Accent color: Dark gold (#B8860B) for highlights and important information.
- Body and headline font: 'Inter' sans-serif for a clean and modern look. Suitable for information display and user interface.
- Use cricket-themed icons for navigation and match information.
- Clear separation of admin and user interfaces for ease of use.
- Subtle animations for score updates and match events.
